ansible-brew
======

Installs packages I use via homebrew. Please fork and adjust to your liking.

Requirements
------------

This role requires Ansible 1.9 or higher and OS X and homebrew installed.

Role Variables
--------------


Example Playbook
----------------

- hosts: all  
  roles:
    - vglen.ansible-brew
```
